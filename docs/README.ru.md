## Список функций

### Практика слов

- Режимы практики: Следование / Диктант / Самопроверка / Письмо по памяти
- Умный режим: Автоматически рассчитывает изучаемые слова на основе кривых памяти, углубляя память через диктант
- Свободный режим: Без ограничений, планируйте своё обучение самостоятельно
- Предоставляет фонетику, произношение (американское/британское), примеры предложений, фразы, синонимы, однокоренные слова, этимологию, статистику ошибок и многое другое

### Запоминание статей

- Встроенные классические учебники; вы также можете добавлять или импортировать статьи с переводом в один клик и двуязычным сравнением
- Двойные режимы следования + диктант, ввод предложение за предложением с автоматическим произношением для более эффективного запоминания
- Поддерживает прослушивание во время письма по памяти для закрепления обучения

### Избранное, Ошибочные слова, Освоенные

- Слова, набранные неправильно во время обучения, автоматически добавляются в вашу тетрадь ошибочных слов для последующего повторения
- Активно добавляйте слова в освоенные, чтобы автоматически пропускать их в будущих сессиях
- Добавляйте слова в избранное для закрепления и повторения

### Высокая настраиваемость

- Богатые звуковые эффекты клавиатуры
- Настраиваемые горячие клавиши
- Высококонфигурируемые параметры настройки

### Чистый и эффективный

- Чистый дизайн, современный интерфейс, без рекламы
- Освежающий интерфейс, простое управление
- Без принудительной подписки на какую-либо платформу

### Библиотека словарного запаса

Встроенный часто используемый словарный запас, включая CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, английский для аспирантуры, TEM-4, TEM-8 и многое другое.
Разработан для удовлетворения потребностей большинства пользователей в изучении словарного запаса. Приветствуются дополнительные словарные вклады от сообщества.

## Запуск проекта

#### Примечание: Этот проект может работать автономно с данными, сохранёнными локально. Ручное резервное копирование требуется при смене устройств; это не влияет на нормальное использование.
Этот проект построен на `Nuxt` и требует среды Node.js.

1. Установите NodeJS, см. [официальную документацию](https://nodejs.org/en/download)
2. Проект большой. Рекомендуется использовать `git clone --depth 1 https://github.com/zyronon/TypeWords.git` для клонирования только последнего коммита. Функция Download ZIP GitHub не будет работать правильно.
3. В корневом каталоге проекта откройте терминал и выполните `pnpm install` для загрузки зависимостей.
4. Выполните `pnpm run dev` для запуска проекта. Адрес по умолчанию [`http://localhost:5567`](http://localhost:5567)
5. Откройте [`http://localhost:5567`](http://localhost:5567) в браузере для доступа к проекту.
6. Выполните `pnpm run generate` для сборки файлов проекта.
