## Liste des fonctionnalités

### Pratique des mots

- Modes de pratique : Suivi / Dictée / Auto-test / Écriture de mémoire
- Mode intelligent : Calcule automatiquement les mots d'apprentissage basés sur les courbes de mémoire, approfondissant la mémoire par la dictée
- Mode libre : Sans restrictions, planifiez votre propre apprentissage
- Fournit la phonétique, la prononciation (américaine/britannique), des phrases d'exemple, des expressions, des synonymes, des mots de même racine, l'étymologie, les statistiques d'erreurs et plus

### Mémorisation d'articles

- Manuels classiques intégrés ; vous pouvez également ajouter ou importer des articles avec traduction en un clic et comparaison bilingue
- Modes doubles suivi + dictée, saisie phrase par phrase avec prononciation automatique pour une mémorisation plus efficace
- Prend en charge l'écoute pendant l'écriture de mémoire pour renforcer l'apprentissage

### Favoris, Mots erronés, Maîtrisés

- Les mots mal tapés pendant l'apprentissage sont automatiquement ajoutés à votre carnet de mots erronés pour révision ultérieure
- Ajoutez activement des mots aux maîtrisés pour les ignorer automatiquement dans les sessions futures
- Ajoutez des mots aux favoris pour consolider et réviser

### Hautement personnalisable

- Riches effets sonores de clavier
- Raccourcis personnalisables
- Options de configuration hautement configurables

### Propre et efficace

- Design épuré, interface moderne, sans publicité
- Interface rafraîchissante, fonctionnement simple
- Pas d'abonnement forcé à une plateforme

### Bibliothèque de vocabulaire

Vocabulaire couramment utilisé intégré incluant CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, anglais pour études supérieures, TEM-4, TEM-8 et plus.
Conçu pour répondre aux besoins d'apprentissage du vocabulaire de la plupart des utilisateurs. Les contributions de vocabulaire supplémentaire de la communauté sont les bienvenues.

## Exécuter le projet

#### Note : Ce projet peut fonctionner de manière autonome avec des données enregistrées localement. Une sauvegarde manuelle est requise lors du changement d'appareil ; cela n'affecte pas l'utilisation normale.
Ce projet est construit avec `Nuxt` et nécessite un environnement Node.js.

1. Installez NodeJS, référez-vous à la [documentation officielle](https://nodejs.org/en/download)
2. Le projet est volumineux. Il est recommandé d'utiliser `git clone --depth 1 https://github.com/zyronon/TypeWords.git` pour cloner uniquement le dernier commit. La fonction Download ZIP de GitHub ne fonctionnera pas correctement.
3. Dans le répertoire racine du projet, ouvrez un terminal et exécutez `pnpm install` pour télécharger les dépendances.
4. Exécutez `pnpm run dev` pour démarrer le projet. L'adresse par défaut est [`http://localhost:5567`](http://localhost:5567)
5. Ouvrez [`http://localhost:5567`](http://localhost:5567) dans votre navigateur pour accéder au projet.
6. Exécutez `pnpm run generate` pour construire les fichiers du projet.
