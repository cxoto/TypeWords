## 機能一覧

### 単語練習

- 練習モード：フォロー入力 / ディクテーション / セルフテスト / 暗記
- スマートモード：記憶曲線に基づいて学習単語を自動計算し、ディクテーションで記憶を深める
- フリーモード：制限なし、自分で計画
- 発音記号、発音（アメリカ英語/イギリス英語）、例文、フレーズ、類義語、同根語、語源、エラー統計などの機能を提供

### 文章暗記

- クラシックな教科書を内蔵、記事の追加・インポートも可能、ワンクリック翻訳・対訳機能付き
- フォロー入力 + ディクテーションのデュアルモード、文ごとの入力、自動発音でより効率的な暗記
- 聞きながら書く機能で記憶を強化

### お気に入り、間違いノート、習得済み

- 学習中に入力ミスした単語は自動的に間違いノートに追加され、後で復習しやすい
- 習得済みに追加すると、以降の学習で自動的にスキップ
- お気に入りに追加して復習を強化

### 高度なカスタマイズ

- 豊富なキーボード効果音
- カスタマイズ可能なショートカットキー
- 高度にカスタマイズ可能な設定オプション

### シンプルで効率的

- シンプルなデザイン、モダンなUI、広告なし
- すっきりしたインターフェース、簡単操作
- プラットフォームへの強制フォローなし

### 単語帳

CET-4、CET-6、GMAT、GRE、IELTS、SAT、TOEFL、大学院英語、専門英語4級、専門英語8級などの一般的な単語帳を内蔵。
ほとんどのユーザーの単語学習ニーズを満たすよう設計されています。コミュニティからの単語帳の貢献も大歓迎です。

## 実行方法

#### 注：このプロジェクトは単独で実行でき、データはローカルに保存されます。デバイスを変更する場合は手動でバックアップが必要ですが、通常の使用には影響しません。
このプロジェクトは `Nuxt` で構築されており、Node.js 環境が必要です。

1. NodeJS をインストール、[公式ドキュメント](https://nodejs.org/en/download)を参照
2. プロジェクトファイルが大きいため、`git clone --depth 1 https://github.com/zyronon/TypeWords.git` コマンドで最新のコミットのみをクローンすることをお勧めします。GitHub の Download ZIP 機能では正常に動作しません。
3. プロジェクトのルートディレクトリでターミナルを開き、`pnpm install` を実行して依存関係をダウンロード。
4. `pnpm run dev` を実行してプロジェクトを起動。デフォルトアドレスは [`http://localhost:5567`](http://localhost:5567)
5. ブラウザで [`http://localhost:5567`](http://localhost:5567) を開いてプロジェクトにアクセス。
6. `pnpm run generate` を実行してプロジェクトファイルをビルド。
