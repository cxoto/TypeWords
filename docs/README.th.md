## รายการฟีเจอร์

### ฝึกคำศัพท์

- โหมดฝึก: ตามพิมพ์ / เขียนตามคำบอก / ทดสอบตัวเอง / เขียนจากความจำ
- โหมดอัจฉริยะ: คำนวณคำศัพท์ที่เรียนอัตโนมัติตามเส้นโค้งความจำ เพิ่มความจำผ่านการเขียนตามคำบอก
- โหมดอิสระ: ไม่มีข้อจำกัด วางแผนการเรียนรู้ด้วยตัวเอง
- มีสัทอักษร การออกเสียง (อเมริกัน/อังกฤษ) ประโยคตัวอย่าง วลี คำพ้องความหมาย คำรากเดียวกัน นิรุกติศาสตร์ สถิติข้อผิดพลาด และอื่นๆ

### ท่องบทความ

- มีหนังสือเรียนคลาสสิกในตัว; คุณยังสามารถเพิ่มหรือนำเข้าบทความพร้อมการแปลคลิกเดียวและการเปรียบเทียบสองภาษา
- โหมดคู่ตามพิมพ์ + เขียนตามคำบอก ป้อนทีละประโยค ออกเสียงอัตโนมัติเพื่อการท่องจำที่มีประสิทธิภาพมากขึ้น
- รองรับการฟังขณะเขียนจากความจำเพื่อเสริมการเรียนรู้

### รายการโปรด คำผิด คำที่เชี่ยวชาญแล้ว

- คำที่พิมพ์ผิดระหว่างเรียนจะถูกเพิ่มลงในสมุดคำผิดอัตโนมัติเพื่อทบทวนในภายหลัง
- เพิ่มคำลงในรายการเชี่ยวชาญแล้วเพื่อข้ามอัตโนมัติในเซสชันถัดไป
- เพิ่มคำลงในรายการโปรดเพื่อทบทวนและเสริมความแข็งแกร่ง

### ปรับแต่งได้สูง

- เอฟเฟกต์เสียงคีย์บอร์ดหลากหลาย
- ปุ่มลัดที่ปรับแต่งได้
- ตัวเลือกการตั้งค่าที่กำหนดค่าได้สูง

### สะอาดและมีประสิทธิภาพ

- ดีไซน์สะอาด UI ทันสมัย ไม่มีโฆษณา
- อินเทอร์เฟซสดชื่น ใช้งานง่าย
- ไม่มีการบังคับสมัครสมาชิกแพลตฟอร์มใดๆ

### คลังคำศัพท์

มีคำศัพท์ที่ใช้บ่อยในตัว รวมถึง CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, ภาษาอังกฤษบัณฑิตศึกษา, TEM-4, TEM-8 และอื่นๆ
ออกแบบมาเพื่อตอบสนองความต้องการการเรียนรู้คำศัพท์ของผู้ใช้ส่วนใหญ่ ยินดีต้อนรับการสนับสนุนคำศัพท์เพิ่มเติมจากชุมชน

## การรันโปรเจกต์

#### หมายเหตุ: โปรเจกต์นี้สามารถรันแบบสแตนด์อโลนโดยข้อมูลจะถูกบันทึกในเครื่อง ต้องสำรองข้อมูลด้วยตนเองเมื่อเปลี่ยนอุปกรณ์; สิ่งนี้ไม่ส่งผลกระทบต่อการใช้งานปกติ
โปรเจกต์นี้สร้างด้วย `Nuxt` และต้องการสภาพแวดล้อม Node.js

1. ติดตั้ง NodeJS ดู[เอกสารอย่างเป็นทางการ](https://nodejs.org/en/download)
2. โปรเจกต์มีขนาดใหญ่ แนะนำให้ใช้ `git clone --depth 1 https://github.com/zyronon/TypeWords.git` เพื่อโคลนเฉพาะคอมมิตล่าสุด ฟังก์ชัน Download ZIP ของ GitHub จะไม่ทำงานอย่างถูกต้อง
3. ในไดเรกทอรีรากของโปรเจกต์ เปิดเทอร์มินัลและรัน `pnpm install` เพื่อดาวน์โหลด dependencies
4. รัน `pnpm run dev` เพื่อเริ่มโปรเจกต์ ที่อยู่เริ่มต้นคือ [`http://localhost:5567`](http://localhost:5567)
5. เปิด [`http://localhost:5567`](http://localhost:5567) ในเบราว์เซอร์เพื่อเข้าถึงโปรเจกต์
6. รัน `pnpm run generate` เพื่อบิลด์ไฟล์โปรเจกต์

## ฟีเจอร์และข้อเสนอแนะ

โปรเจกต์อยู่ในช่วงพัฒนาเริ่มต้น ฟีเจอร์ใหม่กำลังถูกเพิ่มอย่างต่อเนื่อง หากคุณมีข้อเสนอแนะหรือคำขอฟีเจอร์ อย่าลังเลที่จะเปิด `Issue`
หากคุณชอบปรัชญาการออกแบบของซอฟต์แวร์นี้ กรุณาส่ง `PR` ขอบคุณสำหรับการสนับสนุนของคุณ!
