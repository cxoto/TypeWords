## Список функцій

### Практика слів

- Режими практики: Слідування / Диктант / Самоперевірка / Письмо з пам'яті
- Розумний режим: Автоматично розраховує слова для вивчення на основі кривих пам'яті, поглиблюючи запам'ятовування через диктант
- Вільний режим: Без обмежень, плануйте своє власне навчання
- Надає фонетику, вимову (американську/британську), приклади речень, фрази, синоніми, однокореневі слова, етимологію, статистику помилок та багато іншого

### Запам'ятовування статей

- Вбудовані класичні підручники; ви також можете додавати або імпортувати статті з перекладом в один клік та двомовним порівнянням
- Подвійні режими слідування + диктант, введення речення за реченням з автоматичною вимовою для ефективнішого запам'ятовування
- Підтримує слухання під час письма з пам'яті для закріплення навчання

### Обране, Помилкові слова, Освоєні

- Слова, набрані неправильно під час навчання, автоматично додаються до вашого зошита помилкових слів для подальшого повторення
- Активно додавайте слова до освоєних, щоб автоматично пропускати їх у майбутніх сесіях
- Додавайте слова до обраного для закріплення та повторення

### Висока налаштовуваність

- Багаті звукові ефекти клавіатури
- Налаштовувані гарячі клавіші
- Висококонфігуровані параметри налаштування

### Чистий та ефективний

- Чистий дизайн, сучасний інтерфейс, без реклами
- Освіжаючий інтерфейс, проста робота
- Без примусової підписки на будь-яку платформу

### Бібліотека словникового запасу

Вбудований часто використовуваний словниковий запас, включаючи CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, англійську для аспірантури, TEM-4, TEM-8 та багато іншого.
Розроблено для задоволення потреб більшості користувачів у вивченні словникового запасу. Вітаються додаткові словникові внески від спільноти.

## Запуск проекту

#### Примітка: Цей проект може працювати автономно з даними, збереженими локально. Ручне резервне копіювання потрібне при зміні пристроїв; це не впливає на нормальне використання.
Цей проект побудований на `Nuxt` і вимагає середовища Node.js.

1. Встановіть NodeJS, див. [офіційну документацію](https://nodejs.org/en/download)
2. Проект великий. Рекомендується використовувати `git clone --depth 1 https://github.com/zyronon/TypeWords.git` для клонування лише останнього коміту. Функція Download ZIP GitHub не працюватиме правильно.
3. У кореневому каталозі проекту відкрийте термінал і виконайте `pnpm install` для завантаження залежностей.
4. Виконайте `pnpm run dev` для запуску проекту. Адреса за замовчуванням [`http://localhost:5567`](http://localhost:5567)
5. Відкрийте [`http://localhost:5567`](http://localhost:5567) у браузері для доступу до проекту.
6. Виконайте `pnpm run generate` для збірки файлів проекту.
