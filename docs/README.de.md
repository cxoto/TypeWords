## Funktionsliste

### Wortübungen

- Übungsmodi: Nachschreiben / Diktat / Selbsttest / Auswendigschreiben
- Intelligenter Modus: Berechnet automatisch Lernwörter basierend auf Gedächtniskurven und vertieft das Gedächtnis durch Diktat
- Freier Modus: Keine Einschränkungen, plane dein eigenes Lernen
- Bietet Phonetik, Aussprache (amerikanisch/britisch), Beispielsätze, Phrasen, Synonyme, Stammwörter, Etymologie, Fehlerstatistiken und mehr

### Artikelauswendiglernen

- Eingebaute klassische Lehrbücher; du kannst auch Artikel hinzufügen oder importieren mit Ein-Klick-Übersetzung und zweisprachigem Vergleich
- Duale Modi Nachschreiben + Diktat, Satz-für-Satz-Eingabe mit automatischer Aussprache für effizienteres Auswendiglernen
- Unterstützt Hören beim Schreiben aus dem Gedächtnis zur Verstärkung des Lernens

### Favoriten, Fehlerwörter, Gemeistert

- Falsch eingegebene Wörter während des Lernens werden automatisch zu deinem Fehlerwörterbuch hinzugefügt zur späteren Wiederholung
- Füge Wörter aktiv zu "Gemeistert" hinzu, um sie in zukünftigen Sitzungen automatisch zu überspringen
- Füge Wörter zu Favoriten hinzu zur Konsolidierung und Wiederholung

### Hochgradig anpassbar

- Reichhaltige Tastatur-Soundeffekte
- Anpassbare Tastenkürzel
- Hochgradig konfigurierbare Einstellungsoptionen

### Sauber und effizient

- Sauberes Design, moderne Benutzeroberfläche, werbefrei
- Erfrischende Oberfläche, einfache Bedienung
- Kein erzwungenes Abonnement irgendeiner Plattform

### Vokabelbibliothek

Eingebautes häufig verwendetes Vokabular einschließlich CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, Postgraduierten-Englisch, TEM-4, TEM-8 und mehr.
Entwickelt, um die Vokabellernbedürfnisse der meisten Benutzer zu erfüllen. Community-Beiträge zusätzlicher Vokabeln sind willkommen.

## Das Projekt ausführen

#### Hinweis: Dieses Projekt kann eigenständig ausgeführt werden, wobei Daten lokal gespeichert werden. Manuelles Backup ist beim Gerätewechsel erforderlich; dies beeinflusst die normale Nutzung nicht.
Dieses Projekt ist mit `Nuxt` gebaut und erfordert eine Node.js-Umgebung.

1. Installiere NodeJS, siehe [offizielle Dokumentation](https://nodejs.org/en/download)
2. Das Projekt ist groß. Es wird empfohlen, `git clone --depth 1 https://github.com/zyronon/TypeWords.git` zu verwenden, um nur den letzten Commit zu klonen. Die Download ZIP-Funktion von GitHub wird nicht richtig funktionieren.
3. Im Projektstammverzeichnis öffne ein Terminal und führe `pnpm install` aus, um Abhängigkeiten herunterzuladen.
4. Führe `pnpm run dev` aus, um das Projekt zu starten. Die Standardadresse ist [`http://localhost:5567`](http://localhost:5567)
5. Öffne [`http://localhost:5567`](http://localhost:5567) in deinem Browser, um auf das Projekt zuzugreifen.
6. Führe `pnpm run generate` aus, um die Projektdateien zu bauen.
