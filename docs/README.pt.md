## Lista de recursos

### Prática de palavras

- Modos de prática: Acompanhamento / Ditado / Autoteste / Escrita de memória
- Modo inteligente: Calcula automaticamente as palavras de aprendizado com base em curvas de memória, aprofundando a memória através do ditado
- Modo livre: Sem restrições, planeje seu próprio aprendizado
- Fornece fonética, pronúncia (americana/britânica), frases de exemplo, expressões, sinônimos, palavras com mesma raiz, etimologia, estatísticas de erros e mais

### Memorização de artigos

- Livros didáticos clássicos integrados; você também pode adicionar ou importar artigos com tradução com um clique e comparação bilíngue
- Modos duplos de acompanhamento + ditado, entrada frase por frase com pronúncia automática para memorização mais eficiente
- Suporta ouvir enquanto escreve de memória para reforçar o aprendizado

### Favoritos, Palavras erradas, Dominadas

- Palavras digitadas incorretamente durante o aprendizado são automaticamente adicionadas ao seu caderno de palavras erradas para revisão posterior
- Adicione ativamente palavras às dominadas para pulá-las automaticamente em sessões futuras
- Adicione palavras aos favoritos para consolidar e revisar

### Altamente personalizável

- Ricos efeitos sonoros de teclado
- Atalhos personalizáveis
- Opções de configuração altamente configuráveis

### Limpo e eficiente

- Design limpo, interface moderna, sem anúncios
- Interface refrescante, operação simples
- Sem assinatura forçada de qualquer plataforma

### Biblioteca de vocabulário

Vocabulário de uso comum integrado incluindo CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, inglês para pós-graduação, TEM-4, TEM-8 e mais.
Projetado para atender às necessidades de aprendizado de vocabulário da maioria dos usuários. Contribuições de vocabulário adicional da comunidade são bem-vindas.

## Executando o projeto

#### Nota: Este projeto pode ser executado de forma independente com dados salvos localmente. Backup manual é necessário ao trocar de dispositivos; isso não afeta o uso normal.
Este projeto é construído com `Nuxt` e requer um ambiente Node.js.

1. Instale o NodeJS, consulte a [documentação oficial](https://nodejs.org/en/download)
2. O projeto é grande. Recomenda-se usar `git clone --depth 1 https://github.com/zyronon/TypeWords.git` para clonar apenas o último commit. A função Download ZIP do GitHub não funcionará corretamente.
3. No diretório raiz do projeto, abra um terminal e execute `pnpm install` para baixar as dependências.
4. Execute `pnpm run dev` para iniciar o projeto. O endereço padrão é [`http://localhost:5567`](http://localhost:5567)
5. Abra [`http://localhost:5567`](http://localhost:5567) no seu navegador para acessar o projeto.
6. Execute `pnpm run generate` para compilar os arquivos do projeto.
